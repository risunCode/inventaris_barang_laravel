## ğŸ”§ Advanced Customization

### 1. Custom Fields for Commodities
Create migration:

```bash
php artisan make:migration add_custom_fields_to_commodities_table
```

```php
// database/migrations/xxxx_add_custom_fields_to_commodities_table.php
public function up()
{
    Schema::table('commodities', function (Blueprint $table) {
        $table->string('custom_field_1')->nullable();
        $table->text('custom_field_2')->nullable();
        $table->decimal('custom_field_3', 10, 2)->nullable();
    });
}
```

Update `app/Models/Commodity.php`:

```php
protected $fillable = [
    // existing fields...
    'custom_field_1',
    'custom_field_2', 
    'custom_field_3',
];
```

### 2. Custom Validation Rules
Edit `app/Http/Requests/StoreCommodityRequest.php`:

```php
public function rules()
{
    return [
        'name' => 'required|string|max:255',
        'custom_field_1' => 'required|string|max:100',
        'custom_field_2' => 'nullable|string|max:500',
        'custom_field_3' => 'nullable|numeric|min:0',
    ];
}
```

```php
public function boot()
{
    Password::defaults(function () {
        return Password::min(8)
            ->mixedCase()
            ->numbers()
            ->symbols()
            ->uncompromised();
    });
}
```

## ğŸ“Š Custom Reports


Edit `app/Http/Controllers/ReportController.php`:

```php
public function customReport()
{
    $data = Commodity::where('custom_field', 'value')->get();
    $pdf = PDF::loadView('reports.pdf.custom-report', compact('data'));
```

